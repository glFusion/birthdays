glFusion Birthdays Plugin Configuration

The Birthdays Plugin provides an improved search engind for glFusion that indexes documents and returns results weighted by relevance.

Main Options

Date Display Format
Enter the format string to be used when displaying dates. Refer to the PHP date() function for more information
Login required to view?
If this is True, then anonymous uses will not be able to view birthdays.
Enable login message?
If this is True then a popup "Happy Birthday" message will be displayed when a user logs in on his or her birthday.
Enable Birthday Subscriptions
Select "True" to allow site users to subscribe to birthday notifications for other users. Requires glFusion version 1.7.4+.
Enable Birthday Cards
Select "True" to have "Happy Birthday" messages sent to site uses on their birthday.
